# Student Management System 

A robust desktop application to manage student records efficiently, built using Java Swing for the GUI, JDBC for database connectivity, and MySQL as the backend. Perfect for schools, colleges, or anyone learning Java desktop application development.

Features 

Student Management – Add, update, view, and delete student records.

Attendance Tracking – Keep track of student attendance efficiently.

Course & Grade Management – Assign courses and manage grades seamlessly.

Search & Filter – Quickly find students by name, ID, or course.

Database Integration – Fully integrated with MySQL for persistent storage.

User-Friendly GUI – Intuitive and responsive interface built with Swing.

# Technologies Used :

Java – Core application logic

Swing – Desktop GUI design

JDBC – Database connectivity

MySQL – Relational database management
